Accesibilidad digital

Declaración de accesibilidad

En PI - Pisos & Inmuebles trabajamos para que nuestra página web sea accesible para el mayor número posible de personas, con independencia de sus capacidades, dispositivo o entorno tecnológico. Esta declaración de accesibilidad se basa en las WCAG 2.1 del W3C – Web Accessibility Initiative (WAI).

Etiquetado semántico

Toda la estructura del sitio se ha desarrollado empleando etiquetas semánticas de HTML5 (<header>, <nav>, <main>, <section>, <article> y <footer>) para mejorar la comprensión por parte de lectores de pantalla y navegadores especializados.

Además, se han utilizado atributos aria-label y alt en todos los elementos visuales y enlaces relevantes, de forma que el contenido sea interpretado correctamente por tecnologías de asistencia.

Texto alternativo de las imágenes

Todas las imágenes incorporan un texto alternativo (alt) que describe su contenido o función. Las imágenes puramente decorativas se han omitido del marcado HTML o gestionado mediante CSS, para evitar distracciones o redundancias en la lectura de pantalla.

Uso correcto de los colores

Los colores empleados respetan la identidad visual del sitio y cumplen con los niveles de contraste recomendados por las pautas WCAG 2.1 (nivel AA). Además, se han implementado estilos alternativos que permiten aumentar el contraste o emplear fondos oscuros para reducir la fatiga visual.

Activación de estilos accesibles

El sitio ofrece varias hojas de Estilo accesibles que pueden seleccionarse desde el navegador web. En Mozilla Firefox, el usuario puede activar estas opciones desde el menú Ver → Estilo de página:

  • Modo noche: interfaz con fondo oscuro para lectura nocturna.
  • Alto contraste: maximiza la legibilidad mediante colores muy diferenciados.
  • Texto grande: aumenta el tamaño del texto para usuarios con baja visión.
  • Alto contraste y texto grande: combina ambas opciones anteriores.
  • Estilo para impresión: aplicado automáticamente al imprimir o generar PDF.

En navegadores que no soportan la selección de estilos alternativos (como Chrome o Edge), se puede utilizar el menú de accesibilidad propio del sitio o las extensiones del navegador para modificar el contraste o el tamaño del texto.

Otras características de accesibilidad

  • Diseño adaptativo y responsive para diferentes dispositivos y tamaños de pantalla.
  • Compatibilidad con ampliación mediante zoom sin pérdida de funcionalidad.
  • Separación entre contenido y presentación mediante hojas de Estilo CSS.
  • Enlaces y botones accesibles con teclado y visibles en su estado de foco.
  • Uso de tipografía Atkinson Hyperlegible en los estilos accesibles para mejorar la lectura.

Compromiso de mejora continua

Nuestro equipo sigue trabajando para mejorar la accesibilidad de este sitio web. Si encuentra alguna dificultad para acceder a la información o funcionalidades, puede ponerse en contacto con nosotros a través del formulario de contacto o mediante correo electrónico a correo@ua.es.
